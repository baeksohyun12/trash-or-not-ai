# 피그마에서 모듈화?

Select: Design

### 기존 방식

피그마로 디자인 하고 재활용품별 결과 화면을 만들 때 Ctrl +D를 이용하여 복사하여 이미지, 텍스트 등을 하나하나 바꿨다.

### 문제 상황

기존 방식을 사용하다 보니 확실히 시간이 오래 걸리고 위치도 하나씩 다 맞춰야 해서 불편했다. 플라스틱을 처음으로 만들고 캔/고철류, 비닐, 스티로폼, 일반쓰레기, 병류, 종이류 7개의 화면을 더 만들면서 단순 노동이지만 힘들었다. 재활용품별로 텍스트 색, 글자 수, 이미지, 유의사항 문구가 다 다르다보니 색과 이미지를 바꾸는 것 뿐만 아니라 정렬하는 것도 일이었다. 

### 해결 방안

피그마에서는 컴포넌트 모듈화가 존재한다. 우선, 컴포넌트는 레고 블럭처럼 복사해서 계속 사용할 수 있고 원본을 수정하면 전부 자동으로 바뀐다.

우선 정렬을 할때는 Auto Layout 기능을 사용하면 항상 가운데 정렬을 유지할 수 있게 할 수 있다. 또한 Color Styles를 사용하면 한 번에 수정이 가능하다고 한다. 

이미지 역시 모듈화가 가능한데, 이미지를 한 컴포넌트 안에 넣어 구조는 고정시키고 내용만 바뀌는 방식으로 구현할 수 있다. 

(하지만 이걸 디자인이 끝나고 알았다는 사실…)

### 결과

피그마로 본격적인 디자인을 하는 건 처음이라서 피그마를 이용하는 방법에서 몰랐던 기능 굉장히 많았던 것 같다. Auto Layout 기능을 진작에 활용했었다면 작업 시간이 줄어들고 다른 것에도 집중할 수 있었을텐데 아쉬움이 남는다. 하지만 이런 새로운 기능을 하나씩 알아가면서 흥미를 느낄 수 있었고 다음부터 디자인 작업할 때 적용할 수 있어 좋은 경험이었다.
